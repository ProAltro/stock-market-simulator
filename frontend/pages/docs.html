Judge0 API Documentation

Reference for creating trading strategies in Python

Data Access Functions

get_ohlcv(symbol, timeframe=None, start_date=None, end_date=None)

Get OHLCV data array for a symbol. Returns a list of dictionaries with keys: time, open, high, low, close, volume.

Technical Indicators

get_sma(symbol, period)

Get Simple Moving Average (SMA) for the specified period.

get_ema(symbol, period)

Get Exponential Moving Average (EMA) for the specified period.

get_rsi(symbol, period=14)

Get Relative Strength Index (RSI). Returns values between 0 and 100.

get_macd(symbol)

Get MACD (Moving Average Convergence Divergence). Returns a dict with macd, signal, and histogram arrays.

get_bollinger(symbol, period=20, std=2)

Get Bollinger Bands. Returns a dict with upper, middle, lower, percentB, and bandwidth arrays.

get_atr(symbol, period=14)

Get Average True Range (ATR) for volatility measurement.

get_stochastic(symbol, k_period=14, d_period=3)

Get Stochastic Oscillator. Returns a dict with k and d arrays.

get_adx(symbol, period=14)

Get Average Directional Index (ADX). Returns a dict with adx, plusDI, and minusDI.

Trading Functions

buy(symbol, quantity, price=None)

Buy shares of a symbol. If price is not specified, uses the current close price.

sell(symbol, quantity, price=None)

Sell shares of a symbol. If price is not specified, uses the current close price.

Account & Position Info

get_cash()

Get current available cash balance.

get_positions()

Get dictionary of all current positions {symbol: quantity}.

get_position(symbol)

Get current quantity held for a specific symbol. Returns 0 if no position.

get_price(symbol)

Get the latest known price for a symbol.
